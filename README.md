# SimpleDsa

A simple collection of data structures for anyone who wants to expand upon them or simply use them as is!




## Features

- Linked List (Single and Double)

## Features in Progress

- Trie
- Trees (BST)
- Graph
- Hash Table
- Stacks & Queues
- And many variations of the above!
